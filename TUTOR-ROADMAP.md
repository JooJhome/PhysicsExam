- **โหมดทำข้อสอบ**: จับเวลา/ไม่จับเวลา/แบบฝึกหัด (ทำซ้ำ), แบบสอบถามหลังสอบ, ดูเฉลยครั้งเดียว
- **หน้านักเรียน**: การ์ดสะท้อนสถานะ (ยังไม่ทำ/ทำค้าง/ทำเสร็จ), จัดกลุ่ม "ต้องทำ/ทำเสร็จแล้ว", ProgressRing, dialog ยืนยันก่อนเริ่ม, แสดงช่วงวันเปิด/ปิด (เฉพาะวัน)
- **UX ทั่วระบบ**: loading skeletons ทุกหน้า + สปินเนอร์เมนูตอนกำลังพาไป

## Phase 1 — ความถูกต้อง & กันข้อมูลหาย ⭐ (ทำก่อนสุด)
"พลาดแล้วเจ็บ" — กระทบความเชื่อใจ/ข้อมูลจริง สำคัญกว่าฟีเจอร์ใหม่

### 1.1 แก้เฉลย + คิดคะแนนใหม่ (fix key + re-grade) · M 🗄️
- **ทำไม**: หน้าผลสอบมี banner "คะแนนต่ำผิดปกติ · ตรวจ mapping เฉลย" แต่กดได้แค่ **ดู แก้ไม่ได้** → ถ้าเฉลยผิด คะแนนทุกคนผิดและติวเตอร์ติดตาย (เคยต้องแก้ผ่าน SQL)
- **ขอบเขต**: แก้เฉลยรายข้อ → re-grade attempts ที่ส่งแล้วอัตโนมัติ (คง exam id + ประวัติ)

### 1.2 autosave คำตอบระหว่างทำ · M 🗄️
- **ทำไม**: ExamRunner เซฟตอนกดส่งครั้งเดียว → เน็ตหลุด/ปิดเบราว์เซอร์ = คำตอบหายหมด (และเป็นเหตุที่หน้านักเรียนโชว์ "เหลือกี่ข้อ" ไม่ได้)
- **ขอบเขต**: เซฟคำตอบเป็นช่วงๆ ลง `attempts.answers` + ปลดล็อก "เหลือกี่ข้อ" ในหน้านักเรียน

## Phase 2 — ปิด "เร็วๆนี้" / Quick wins
effort ต่ำ เห็นผลเร็ว ปิดเมนูที่ค้างว่า "เร็วๆนี้"

### 2.1 คลังข้อสอบ / archive · S
- DB พร้อม (`exams.status='archived'` ใน migration 0005) เหลือ UI ซ่อน/เรียกคืน

### 2.2 ตั้งค่า/โปรไฟล์ติวเตอร์ · S
- เปลี่ยนรหัส/ชื่อติวเตอร์ — เมนูบัญชีค้าง "เร็วๆนี้"

### 2.3 คัดลอกชุด (duplicate) · S
- copy exam + answer key เป็นชุดใหม่ (draft) — เมนู ⋯ "เร็วๆนี้"

### 2.4 เตือนนักเรียนที่ยังไม่ทำ · S–M
- dashboard เจอ "ยังไม่เริ่ม N คน" แล้วแต่กดอะไรไม่ได้ → ปุ่มคัดลอกข้อความเตือน (ยังไม่มี push/email → clipboard ก่อน)

---

## Phase 3 — ต่อยอดของที่เพิ่งทำ

### 3.1 สรุปคะแนนรายกลุ่ม ในสรุปรายชุด · M
- เฉลี่ย/ผ่าน เทียบห้อง A vs B (ต่อจาก filter กลุ่มที่ทำแล้ว)

### 3.2 live assignment · M
- เพิ่มคนเข้ากลุ่ม → auto-มอบชุดที่กลุ่มเคยได้ (ปัจจุบันเป็น snapshot)

### 3.3 กราฟพัฒนาการรายคนตามเวลา · M
- คะแนน % ข้ามชุดเรียงเวลา ใน drawer รายคน (มี `StudentSummary.trend` เป็นเชื้อ)

---

## Phase 4 — เชิงสอนเชิงลึก (งานใหญ่ คุณค่าสูง)

### 4.1 tag หัวข้อรายข้อ → จุดอ่อนรายเรื่อง · L 🗄️
- "อ่อนกลศาสตร์/ไฟฟ้า" แทน "ข้อที่ N" — ทั้งห้อง + รายคน (survey ถาม free-text อยู่แล้ว)

### 4.2 ฟีดแบ็กติวเตอร์ → นักเรียนต่อ attempt · M 🗄️
- survey มีแค่เด็ก→ครู ยังตอบกลับไม่ได้ → คอมเมนต์ต่อ attempt ให้นักเรียนเห็นในหน้าผล

### 4.3 รายงานรายคน printable/PDF · M
- ส่งผู้ปกครอง — มี CSV gradebook แล้ว ขาดเวอร์ชันรายคน
